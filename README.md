<h2>Getting Started</h2>
<p>The "SunnyBoy" Server manages and stores (weather-)information in its database. Additionally, the Server administers the control of several "MoonY" clients. A "MoonY" requests its to-do-list, if the recieved to-do-ist is completed the server stores the collected data to its database. The client to-do-lists are editable through a web interface. Furthermore, the communication between server and client is done by Rest API endpoints. </p></li>
